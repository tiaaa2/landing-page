Work

Jewelry Website

To better showcase my jewelry brand, I developed a personal website using Java Spring Boot and React. Managing product listing on Instagram proved challenging, so I created this site so it's easier for customers to browse and view prices of available items. The website features admin CRUD operations, allowing for easy management of products.

Biometric Ear Authentication System

I played a key role in implementing ear segmentation and contour detection using YOLOv3, optimizing the process to handle each image in just 0.5 seconds. Additionally, I contributed to the development of a web-based system designed to manage a database of over 100 users, integrating advanced image processing techniques for secure and accurate ear recognition.

About

Creativity has always been a driving force for me, and coding has fascinated me since my school days. Creating solutions and building projects through code feels intuitive to me. I graduated from Rajagiri School of Engineering with a Bachelor of Technology in Computer Science.

My tech interests are varied; I explore machine learning, web development, and any other intriguing fields. I’m always excited to learn and grow my expertise. Beyond technology, I enjoy crafting and selling jewelry, running a thrift store, and traveling.
